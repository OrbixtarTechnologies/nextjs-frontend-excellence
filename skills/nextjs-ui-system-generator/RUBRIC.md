# Rubric

## 3/10
Thin local guidance with generic frontend language.

## 5/10
Some repo awareness, but weak scoping and weak artifact separation.

## 7/10
Useful local frontend asset generation with reasonable boundaries.

## 9/10
Strong repo-aware frontend operating model with justified local assets and clear artifact roles.

## 10/10
A disciplined repo bootstrap skill that inspects the actual Next.js repository, generates only the minimum strong frontend asset set, avoids duplication, and materially improves future Copilot execution quality.
