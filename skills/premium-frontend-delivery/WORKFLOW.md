# Workflow

1. Verify route ownership and UI scope.
2. Verify data source and contract truth.
3. Verify permission-sensitive behavior.
4. Reuse strong component and layout patterns already present.
5. Strengthen visual hierarchy, spacing, and responsiveness.
6. Implement truthful loading, empty, error, success, unauthorized, and not-found states.
7. Validate keyboard accessibility and semantic structure.
8. Produce honest implementation notes using the output contract.
