# UX State Integrity

## Use this skill for
- pages, layouts, forms, lists, dashboards, and detail views
- routes with asynchronous data or permission-sensitive content

## Core mandate
Every meaningful user-facing surface must have truthful state handling.

## Required states
- loading
- empty
- error
- success where applicable
- unauthorized where applicable
- not-found where applicable
