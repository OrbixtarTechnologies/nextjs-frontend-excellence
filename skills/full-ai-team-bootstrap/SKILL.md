# Full AI Team Bootstrap

## Use this skill for
- new repos
- repos with weak or inconsistent Copilot setup
- repos needing a full local AI engineering team system

## Core mandate
Inspect the repository first, then generate only the minimum strong set of repo-local instructions, agents, prompts, skills, and governance docs that the repo actually needs.

## Required inspection
- repo topology
- frameworks and languages
- frontend/backend/data boundaries
- test stack
- security and ops sensitivity
- deployment clues
- repeated workflows
- quality risks
