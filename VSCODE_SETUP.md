# VS Code Workspace Setup

This repository includes a full professional VS Code workspace configuration in `.vscode/`.

## Included
- `settings.json` — formatting, linting, TypeScript, search/file excludes, Copilot, terminal, and UI defaults.
- `extensions.json` — recommended extension pack for Next.js, quality, and productivity.
- `tasks.json` — install/dev/build/lint/typecheck/test commands.
- `launch.json` — server/full-stack/browser debugging profiles.
- `snippets/nextjs.code-snippets` — starter snippets for server components and route handlers.

## Notes
- Task commands use `npm run ...` conventions; adjust if your repo uses pnpm/yarn/bun.
- This setup is designed for a robust, professional Next.js development flow.
