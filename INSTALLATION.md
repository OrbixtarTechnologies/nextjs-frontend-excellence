# Repository Installation Guide

## Goal
Install and operate the `nextjs-uiux-excellence` package from this main repository.

## Steps
1. Clone this repository.
2. Use the package under `plugins/nextjs-uiux-excellence/` in your plugin runtime.
3. Reload your runtime/plugin registry.
4. Verify plugin metadata and hooks:
   - `plugins/nextjs-uiux-excellence/plugin.json`
   - `plugins/nextjs-uiux-excellence/hooks.json`
