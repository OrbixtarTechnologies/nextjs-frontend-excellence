# Contributing

## Contribution policy
- Keep updates aligned to the single merged main branch model.
- Preserve repository-level documentation as the first-class entrypoint.
- Keep plugin package docs in sync with root docs when behavior changes.

## Required updates when changing capabilities
1. Update root docs (`README.md`, `INSTALLATION.md`, `USE_CASES.md`, `ARCHITECTURE.md`).
2. Update plugin docs and manifests.
3. Update changelog/version notes.
