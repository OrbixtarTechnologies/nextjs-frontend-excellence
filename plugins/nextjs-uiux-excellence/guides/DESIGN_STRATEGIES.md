# Broader Design Strategies

## Strategy 1: Brand-Driven Premium System
- High-contrast hierarchy, strong typography scale, opinionated spacing rhythm.
- Best fit: startup landing + premium SaaS.

## Strategy 2: Enterprise Utility System
- Dense information layout, predictable controls, low-surprise interaction patterns.
- Best fit: admin portals, operations tools, B2B platforms.

## Strategy 3: Editorial + Product Blend
- Narrative storytelling plus conversion-focused application surfaces.
- Best fit: docs-first products, knowledge SaaS, media + app combinations.

## Selection matrix
Choose strategy using:
1. Primary user intention (discover vs operate vs analyze).
2. Data density requirements.
3. Brand expression tolerance.
4. Accessibility and localization needs.

## Implementation rule
When users request a broad redesign, choose one strategy explicitly and apply it consistently across layouts, components, and content hierarchy.
