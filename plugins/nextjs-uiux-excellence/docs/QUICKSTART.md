# Quickstart

## 5-minute setup flow
1. Install plugin (see `docs/INSTALLATION.md`).
2. Start with `product-requirements-owner` to define exact scope.
3. Use `nextjs-ui-architect` for UI kit + layout strategy.
4. If request says **revamp**, run `broader-design-architecture-overhaul`.
5. Implement with `nextjs-ui-builder`.
6. Validate with `qa-test-architect` and `principal-code-reviewer`.

## Recommended prompts
- "Revamp whole app with new strategy from scratch."
- "Upgrade only dashboard module to enterprise utility layout."
- "Modernize marketing pages but keep app shell unchanged."
- "Apply 40% redesign: auth + onboarding + settings only."

## Scope interpretation reminders
- whole app = global surfaces + shared systems.
- module = all routes/flows in that module.
- page/section = bounded target only.
- percentage = explicit subset mapping required.
