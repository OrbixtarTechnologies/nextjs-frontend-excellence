# Industry Task Amplification Template

## Input
"<paste the user's raw request here>"

## Amplified Prompt Template
"You are implementing this request at industry-grade quality. Preserve the exact intent, then expand it into a complete production-ready brief.

Deliver:
1. Scope definition and assumptions.
2. Page/feature architecture and layout structure.
3. UI/UX requirements (hierarchy, spacing, typography, responsive breakpoints).
4. Core + advanced + future extension features.
5. Full interaction states (loading/empty/error/success/partial).
6. Accessibility requirements (keyboard, focus, semantics, contrast).
7. Performance constraints and rendering strategy.
8. Acceptance criteria and test plan.
9. Implementation phases with risk controls.

Output must be detailed, actionable, modern, and suitable for direct engineering execution."
