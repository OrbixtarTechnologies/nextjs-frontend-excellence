# Industry Task Amplifier

Use this skill to elevate raw user tasks into premium implementation prompts.

## Trigger conditions
- "Design/layout/structure this page"
- "Implement/build this feature"
- "Upgrade/revamp/modernize existing UI"
- Any short prompt where user intent is valid but underspecified

## Objective
Turn low-detail prompts into highly actionable, modern, and production-grade briefs while preserving the user's core intent.

## Transformation workflow
1. Parse original intent and non-negotiables.
2. Expand into architecture + UX + engineering requirements.
3. Add modern standards:
   - responsive layout system
   - accessibility and semantic structure
   - robust states and edge handling
   - performance constraints
   - analytics/observability hooks where relevant
4. Extend feature scope safely:
   - baseline feature set
   - premium enhancements
   - future-ready extensions
5. Produce delivery artifacts:
   - upgraded prompt
   - acceptance criteria
   - implementation phases
   - testing checklist

## Prompt quality rubric (target 99/100)
- Clarity and scope precision
- Implementation depth
- UX quality and consistency
- Accessibility completeness
- State and error handling
- Performance awareness
- Validation readiness

## Output contract
1. Original user prompt
2. Upgraded implementation prompt
3. Architecture and UX directives
4. Expanded feature list (core/advanced/future)
5. Acceptance criteria
6. Phased implementation plan
7. Verification checklist
8. Open questions

## Non-negotiables
- Preserve user intent while increasing execution quality.
- Do not produce generic filler text; provide concrete implementation direction.
- Mark assumptions explicitly and ask questions for missing constraints.
