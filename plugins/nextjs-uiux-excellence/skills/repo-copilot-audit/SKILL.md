# Repo Copilot Audit

Use this skill to score and improve repo-local Copilot customizations.

## Audit focus
- clarity and enforceability of instructions
- overlap/gap analysis across agents and skills
- evidence of truthful delivery standards
- maintainability and discoverability of plugin assets
