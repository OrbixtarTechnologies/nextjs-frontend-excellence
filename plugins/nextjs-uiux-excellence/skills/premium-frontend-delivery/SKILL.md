# Premium Frontend Delivery

Build premium, truthful, accessible Next.js UI that feels launch-ready.

## Standards
- Use selected UI kit conventions consistently.
- Build reusable professional layouts and route groups.
- Ensure loading/empty/error/success states for every async surface.
- Maintain strong visual hierarchy and responsive polish.


## Revamp trigger handling
When the prompt includes **revamp**, run a redesign-first workflow:
1. Assess current UX/system weaknesses.
2. Propose a distinctly new strategy and layout architecture.
3. Implement replacement-grade UI for requested scope.
4. Report mapping of requested scope vs completed scope.

## Scope handling
- Honor exact scope terms from user language.
- Never silently reduce requested coverage.
- If scope is unclear, explicitly declare chosen interpretation and proceed accordingly.
