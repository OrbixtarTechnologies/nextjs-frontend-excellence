# Next.js Live Docs Policy

## Policy
All version-sensitive implementation guidance must be verified against latest official docs before coding or architectural decisions are finalized.

## Applies to
- Next.js APIs and configuration
- React APIs/patterns
- UI kit usage and breaking changes
- Node ecosystem libraries used in implementation scope

## Required output behavior
- Cite official sources.
- Mention retrieval date/time.
- Mark experimental APIs clearly.
- Call out breaking/deprecated usage.
