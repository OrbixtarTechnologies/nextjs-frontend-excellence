# Repository Architecture

## Single-mainline model
This repo follows a **single merged main branch** approach for canonical plugin evolution.

## Structure
- `plugins/nextjs-uiux-excellence/agents/` — role playbooks
- `plugins/nextjs-uiux-excellence/skills/` — execution systems
- `plugins/nextjs-uiux-excellence/guides/` — strategy + architecture references
- `plugins/nextjs-uiux-excellence/docs/` — plugin-focused operational docs

## Governance
- Scope obedience is mandatory.
- Revamp requests invoke redesign-from-scratch behavior.
- Delivery must be truthful (no fake functionality claims).
