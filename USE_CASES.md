# Repository Use Cases

## Whole-app revamp
Run full redesign from scratch with a new strategy, architecture blueprint, and complete scope adherence.

## Enterprise dashboard modernization
Apply data-dense layout architecture, reusable UI primitives, and governance-driven consistency.

## Partial modernization
Upgrade bounded modules/pages while explicitly reporting out-of-scope surfaces.

## Multi-team delivery
Use agents/skills to coordinate architecture, implementation, QA, security, and documentation.
